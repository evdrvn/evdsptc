evdsptc
=======

sync / async event dispatcher for C/C++

## Getting Started

* cd build
* cmake ..
* make

## Example

* async event
See sync_event_example, test/src/evdsptc_test.cpp 

* sync event
See sync_event_example, test/src/evdsptc_test.cpp 

* and more
See test/src/evdsptc_test.cpp

## Run Tests

* cd test
* sh ./build_cpputest.sh
* make
